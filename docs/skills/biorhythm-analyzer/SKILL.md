# 生物节律分析器

## 概述

生物节律分析器是一个综合工具，用于分析人体生物节律周期并提供个性化的生活建议。该技能结合了传统的生物节律计算与现代天气数据，帮助用户了解自身的生理状态并制定相应的工作和生活计划。

## 核心功能

### 1. 生物节律计算
- 计算体力周期（23天）
- 计算情绪周期（28天）
- 计算智力周期（33天）
- 预测未来7天的节律趋势

### 2. 天气数据集成
- 获取实时天气数据
- 基于温度、湿度、风速等生成穿衣建议
- 根据天气条件提供生活注意事项

### 3. 生活建议生成
- 结合生物节律状态和天气条件
- 提供饮食、运动、工作建议
- 生成个性化报告

### 4. 可视化展示
- 创建交互式图表展示节律数据
- 生成美观的报告页面
- 支持数据导出和分享

## 使用方法

当用户需要分析生物节律时，提供以下信息：
- 出生日期（YYYY-MM-DD格式）
- 目标分析日期（可选，默认为当天）
- 所在城市（用于天气数据）

技能将自动：
1. 计算生物节律数据
2. 获取天气信息
3. 生成综合建议
4. 创建可视化报告

## 技术实现

### 后端计算
- 使用Python进行节律计算
- 集成天气API获取实时数据
- 提供RESTful API接口

### 前端展示
- 基于React + Tailwind CSS
- 使用Chart.js进行数据可视化
- 响应式设计支持多设备

### 数据存储
- 临时会话存储
- 支持数据缓存优化性能

## 集成示例

技能可以集成到现有项目中，作为生物节律分析模块使用。支持自定义样式和功能扩展。
